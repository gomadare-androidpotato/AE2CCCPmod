## AE2CCCPmod
A mixin-based mod that modifies the behavior of AE2 to add spatial storage compatibility with AE2UE v0.56.7 (the latest release at the time of development) and cubicchunks.

### How To Use
**English**
In an environment where [AE2-UE v0.56.7](https://github.com/AE2-UEL/Applied-Energistics-2/releases/tag/v0.56.7) and [CubicChunks v0.0.1271.0](https://www.curseforge.com/minecraft/mc-mods/opencubicchunks/files/5135427) are installed simultaneously in the mods folder,
the crash that occurs when using spatial storage in a cubicchunks environment is fixed simply by adding the latest mixinbooter and this mod release to the mods folder like a normal mod.

Also, a test at Y=-1000 showed no problems.

(Note: While there's a good chance that different versions will work, they are not supported.)
**Japanese**
modsフォルダに[AE2-UEv0.56.7](https://github.com/AE2-UEL/Applied-Energistics-2/releases/tag/v0.56.7)と[CubicChunksv0.0.1271.0](https://www.curseforge.com/minecraft/mc-mods/opencubicchunks/files/5135427)を同時に導入している環境で
最新のmixinbooterとこのmodのリリースをmodsフォルダに普通のmodと同じ様に入れるだけでspatial storageをcubicchunks環境で使った時のクラッシュが修正されます
また、Y=-1000で動作テストしても正常に動作しました

(補足:バージョンが違っても動作する可能性はそこそこ高いがサポートはしない)
